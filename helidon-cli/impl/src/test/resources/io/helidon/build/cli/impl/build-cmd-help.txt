
Usage:	helidon build [OPTIONS] [--project PATH] [--clean] [--mode PLAIN|NATIVE|JLINK]

Build the application

Options:
  -D<name>=<value>    Define a system property
  --verbose           Produce verbose output
  --debug             Produce debug output
  --project           The project directory
  --clean             Perform a clean before the build
  --mode              Build mode (default: PLAIN)
